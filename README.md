# 1vin-zerkalo
1вин зеркало — это рабочий альтернативный домен, позволяющий получить доступ к платформе при недоступности основного сайта. Через актуальное зеркало 1вин можно открыть личный кабинет, перейти в любые разделы и продолжать пользоваться сервисом без ограничений. Зеркала работают стабильно, быстро загружаются и поддерживают любые устройства.
